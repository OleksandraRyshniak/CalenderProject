Mis on Microsoft Project

Microsoft Project on projektijuhtimise tarkvara. See aitab planeerida ülesandeid, jaotada ressursse, jälgida tähtaegu ja kontrollida projekti edenemist.

Mida tähendab "kood" Microsoft Projectis?

Microsoft Projectis võib "kood" tähendada:

  • Ülesannete koodid – unikaalsed identifikaatorid ülesannetele
  • Ressursside koodid – töötajate või materjalide tähistused
  • VBA (makrod) – programmeerimiskood automatiseerimiseks

Näide koodist (VBA makro)

Siin on lihtne näide koodist, mida saab kasutada Microsoft Projectis:

Sub HelloProject() MsgBox "Tervitus Microsoft Projectist!" End Sub

See kood kuvab hüpikakna sõnumiga.
